카페 운영에 맞춘 에너지 진단

일반적인 건물 에너지 분석이 아니라, 카페와 베이커리 카페의 실제 운영 특성을 기준으로 진단합니다.

⚡

전기요금 진단

월 전기요금과 전력사용량을 바탕으로 면적당 사용량, 운영시간당 사용량, 평균 단가를 분석합니다.

📍

주소 기반 날씨 연결

매장 주소를 입력하면 위치를 찾고, 해당 위치의 현재 외기온도, 습도, 풍속을 진단에 반영합니다.

🧊

장비 구성 분석

쇼케이스, 제빙기, 오븐, 냉장고, 에어컨 등 카페 장비 구성을 바탕으로 피크부하를 추정합니다.

🌡️

실내환경 진단

평균 실내온도와 습도를 기준으로 과냉방, 냉방 부족, 습도 관리 필요 여부를 확인합니다.

📊

업종 기준 비교

일반 카페와 베이커리 카페의 기준값을 나누어 같은 업종 대비 전력 사용 수준을 비교합니다.

📝

진단 리포트

핵심 결과와 추천 조치를 텍스트 리포트로 정리해 다운로드할 수 있습니다.

사용 방법은 간단합니다

고지서 정보와 매장 기본 정보만 입력해도 첫 진단을 시작할 수 있습니다.

1

매장 정보 입력

업종, 면적, 운영시간, 월 영업일수를 입력합니다.

2

전기요금 입력

월 전력사용량, 월 전기요금, 계약전력을 입력합니다.

3

주소와 날씨 연결

매장 주소를 기반으로 현재 날씨 데이터를 불러옵니다.

4

진단 결과 확인

점수, 절감 금액, 추천 조치, 리포트를 확인합니다.
